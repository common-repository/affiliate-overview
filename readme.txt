== Description ==

Affiliate-Overview is a Wordpress plugin, which uses the affilinet web services to create an overview over the publisher account on the Wordpress dashboard.

The plugin enables you to display...

* the current account balance (open, confirmed, cancelled)
* the account balance of the last months
* information on the last payments
* and the number of partnerships (active and waiting)
    
...in the wp admin.


== Installation ==


<b>System requirements:</b>

* PHP5 or higher
* SOAP
* Wordpress (tested from version 2.6 onwards)
* an affilinet publisher account
* an affilinet publisher web service password

<b>Installation:</b>

1. Upload the folder `affiliate-overview` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Affiliate-Overview settings and type in your affilinet publisherID and your Webservice-password. 
4. Go back to the dashboard an you'll see the affilinet-account-overview.

== Frequently Asked Questions ==

= Where can I make a affilinet publisher account? =

Sign up at http://www.affili.net

= Where can I get me a affilinet publisher webservice-password? =

* Login to http://publisher.affili.net  
* Go to `Account -> Technical settings -> Webservices`
